# TallerPython_ReyesRamon

## Descripción
Este proyecto contiene una serie de programas en Python que muestran ejemplos prácticos de distintos conceptos fundamentales del lenguaje. Incluye:

- **impresiones.py**: Ejemplos de impresión en pantalla, uso de f-strings, concatenación y captura de datos con `input()`.
- **operaciones.py**: Operaciones aritméticas básicas, calculadora con menú y condicionales.
- **ciclos.py**: Uso de ciclos `for` y `while` con distintos rangos y contadores.
- **listas.py**: Manejo de listas, cálculo de promedios, captura de datos por usuario y actividades de procesamiento de números.
- **archivos.py**: Lectura y manipulación de archivos CSV y Excel, cálculo de promedios, asignación de estatus, gráficos con `matplotlib` y análisis con `NumPy`.

Este repositorio es parte del taller de Python y sirve para reforzar la comprensión de conceptos básicos y su aplicación práctica.

## Clonar el repositorio
Para obtener una copia local del proyecto, ejecuta:


git clone https://github.com/Ramon-Reyes-Del-Angel/TallerPython_ReyesRamon.git

Luego, accede a la carpeta del proyecto:
cd nombre-del-repositorio

Instalación de dependencias
Este proyecto requiere algunas librerías externas. Puedes instalarlas fácilmente usando pip:

pip install -r requirements.txt
Nota: Asegúrate de tener pip actualizado y un entorno de Python 3.8 o superior.

Ejecución del código
Cada archivo se puede ejecutar de forma independiente según la temática que quieras probar. Por ejemplo:

python impresiones.py
python operaciones.py
python ciclos.py
python listas.py
python archivos.py
Sigue las instrucciones que aparezcan en la consola, ya que algunos programas requieren entrada de datos por parte del usuario.
